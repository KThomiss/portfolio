Change 1

I did not have any cta-buttons just plain text with hover effects

Now the users can better tell what they can interact with

Change 2

The size of the headings and text did not flow well and disturbed the hierarchy of the website.

Some text was bigger then headings and some headings was too small.

Also used better html headings for semantics.

Change 3

Changed up the colors to make it more appealing for the target audience.

Added a cool effect when hovering over various headings throughout the page.

Updated how the page responds to differen screensizes on all pages, updating the grid-layout and font-size.

Change 4

Added contact form with validation

Hamburger menu with JavaScript and not CSS where I used the checkbox trick.

Change 5

Fun fact! I went from 1050 CSS code lines in the main css file to under 750 and everything stayed the same. Removing 300+ css lines by being more effecient in my coding is a nice improvement I think.
